在Ubuntu服务器上安装MySQL8

发表于 2020-03-01 更新于 2020-03-28 分类于 软件 ， Linux 阅读次数： Valine：
本文字数： 204 阅读时长 ≈ 1 分钟
因为自己电脑上用的是mysql8，为了部署项目在服务器上的时候不出这方面的问题，服务器就也上mysql8，但是Ubuntu直接apt-get install mysql-server，装的是5.7的版本，所以又得折腾一番

在http://dev.mysql.com/downloads/repo/apt/，下载一个mysql-apt-config_0……._all.deb，传到服务器上，用dpkg -i mysql-apt-config_0._…..all.deb安装执行，选择mysql8，OK之后回到终端窗口

执行sudo apt update

然后使用sudo apt install mysql-server命令，就会安装MySQL8.0
