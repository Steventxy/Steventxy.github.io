操作系统--操作系统的概述

发表于 2020-03-02 更新于 2020-04-27 分类于 学习笔记 ， 操作系统 阅读次数： Valine：

计算机的组成

硬件+软件

引入操作系统的目的

方便性、有效性、可扩充性和开放性

操作系统的作用

  • 是计算机硬件和软件的管理者
  • 是用户使用计算机硬件系统的借口
  • 操作系统是扩展机/虚拟机

操作系统的定义

操作系统是控制和管理计算机系统的软硬件资源、合理地组织计算机工作流程，

以及方便用户的程序和数据集合。

单道批处理系统的特征

  1. 自动性:磁带上的一批作业能自动地逐个地依次运行，无需人工干预

  2. 顺序性:磁带上的各道作业是顺序地进入内存，各道作业的完成顺序与他们进入内存的顺序之间，在正常情况下应完全相同

  3. 单道性:内存中仅有一道程序运行。仅当该程序完成或发生异常情况时才换入其后继程序进入内存运行

操作系统的主要功能

  1. 处理器管理
  2. 存储管理
  3. 设备管理
  4. 文件管理
  5. 用户接口
  6. 网络与通信管理

操作系统基本特征

  1. 并发性

  2. 共享性

    a）互斥使用方式 b）同时使用方式

  3. 虚拟性

  4. 不确定性

    a）“走走停停的执行方式” b）执行结果的不确定性 c）不可预测性
